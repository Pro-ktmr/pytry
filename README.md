# PyTry (パイトライ) | 初心者のための Python 開発環境

PyTry (パイトライ) は初心者のための Python 開発環境です

ご利用はこちらから：https://pro-ktmr.github.io/pytry/

## ローカルへの落とし方

```bash
git clone --recursive git@github.com:Pro-ktmr/pytry.git --depth 1
```

- submodule を使用しているため，`--recursive` が必要です
- 過去に重いデータを直置きしてたため，`--depth 1` を付けないと一部の処理に時間がかかります
